Privacy Policy

DueBar — Last updated June 2026

DueBar is a macOS menu bar application that surfaces your upcoming tasks from Google Tasks and Microsoft To Do in one place. It lets you view, create, complete, and reschedule tasks without leaving your desktop. This policy explains exactly what data the app accesses, how it is used, and how it is protected.

Google Tasks — Data Accessed

DueBar requests access to your Google Tasks via the https://www.googleapis.com/auth/tasks scope. The following specific data fields are accessed:

Data field Where it comes from Why it is accessed
Task list names and IDs Google Tasks API — GET /users/@me/lists To display the list of task lists the user can assign a new task to, and to fetch tasks from each list
Task title Google Tasks API — GET /lists/{listId}/tasks To display each task's name in the menu bar popup
Task due date Google Tasks API — GET /lists/{listId}/tasks To sort and group tasks by urgency (overdue, today, tomorrow, this week, upcoming) and display the due label next to each task
Task notes Google Tasks API — GET /lists/{listId}/tasks To display additional context for a task when the user hovers over the notes icon
Task status Google Tasks API — GET /lists/{listId}/tasks To filter out already-completed tasks so only pending tasks are shown

Google Tasks — Data Usage

DueBar uses Google user data exclusively to provide the task management features described below. No Google user data is used for any other purpose, including advertising, analytics, or AI/ML model training.

Feature Data read Data written
Display tasks Task title, due date, notes, status, list name None
Mark task complete Task ID, list ID Sets status = "completed" via PATCH /lists/{listId}/tasks/{taskId}
Reschedule task Task ID, list ID Updates due date via PATCH /lists/{listId}/tasks/{taskId}
Create task List ID (to know where to create) Posts title, optional notes, and optional due date via POST /lists/{listId}/tasks

Data Storage and Transmission

DueBar does not transmit your data to any server operated by DueBar or any third party. All data flows directly between your Mac and Google's or Microsoft's servers. Specifically:

  • Task data (titles, due dates, notes) is fetched on demand and held only in memory while the app is running. It is never written to disk by DueBar.
  • OAuth access tokens and refresh tokens for Google and Microsoft are stored exclusively in your Mac's secure Keychain. They are never logged, transmitted to DueBar servers, or shared with any third party.
  • Local Tasks data is stored only on your Mac in ~/Library/Application Support/DueBar/local-tasks.json. It never leaves your device.
  • DueBar has no backend, no analytics service, and no crash reporting service. No user data of any kind leaves your device except in API calls made directly to Google or Microsoft on your behalf.

Data Sharing

DueBar does not sell, share, license, or otherwise transfer your data to any third party for any purpose.

Revoking Access

You can sign out at any time from within the app (Settings → Accounts), which immediately removes your tokens from the Keychain.

  • To fully revoke DueBar's access to your Google account, visit myaccount.google.com/permissions.
  • To fully revoke DueBar's access to your Microsoft account, visit account.live.com/consent/Manage.
